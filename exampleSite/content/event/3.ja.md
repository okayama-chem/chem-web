### 内容

工学部（午前）と理学部（午後）の研究室による講義や簡単な実験、研究紹介

| （午前）【工学部】 | 研究室      | タイトル                                                                                     |                                                                                                                                                  |
| --------- | -------- | ---------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| 亀島欣一 先生   | 環境無機材料科学 | エネルギーを生み出すセラミックス材料の実演と解説<br /><img src="/images/events/3/kameshima.png" width="300px" /> | エネルギー関連に利用されるセラミックスを中心とした材料の紹介と種々の発電方法のデモを行います．各種発電方法の有効性と課題について，一緒に考えてみましょう．また，SDGsにおける位置づけも考えてみましょう．                                           |
| 坂倉 彰 先生   | 生物有機化学   | フルーツや清涼剤に含まれる香料の合成<br /><img src="/images/events/3/ester.jpg" width="300px" />           | よく熟れたフルーツは，いい香りがします。その香りの成分は，エステルという構造を持つ有機化合物です。この体験実験では，フルーツや清涼剤の香りを持つ有機化合物の合成を行います。例えば，バナナの香りの成分は，酢酸（お酢の主成分）とアルコール（お酒の仲間）から合成できます。            |
| 大槻高史 先生   | 生体分子工学   | 第三の医薬品として注目 ～ 核酸の合成と分析<br /><img src="/images/events/3/image1-31.png"  width="300px" />  | 生物の設計図であるDNAと、様々な働きをもつRNAは似た構造をしているので、まとめて「核酸」と呼ばれます。コロナワクチンの登場で、核酸が薬になることが最近注目され始めています。そんな核酸を、PCRという手法（下図参照）により増幅・合成し、電気泳動によってその大きさと量を調べてみましょう。 |

| （午後）【理学部】 | 研究室    | タイトル                                                                                        | 概要                                                                                                                                      |
| --------- | ------ | ------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| 松本正和 先生   | 理論化学   | 多面体を手で描く -Polyhedron Challenge- <br /><img src="/images/events/3/poly.png" width="300px" /> | フラーレンに限らず、魅惑的な立体構造を持つ分子はたくさんあります。この講座では、3次元の複雑な図形を、コンピュータグラフィックスを使わず手に描く方法を紹介します。方眼紙の上に、フラーレンや準結晶の立体構造を実際に描いてみましょう。                     |
| 武安伸幸 先生   | 分析化学   | 銀樹の成長と観察<br /><img src="/images/events/3/ag.png"  width="300px" />                          | 条件を整えて銀イオンを還元すると，先端にナノスケールの枝葉を持つ銀樹（写真）が成長します．みなさんには，光学顕微鏡（暗視野）を使って，銀樹が様々な波長（色）の光を捉える様子を観察してもらいます．同様に，金ナノ粒子や銀ナノ粒子も観察して，それらとの違いを比べてみましょう． |
| 西原康師 先生   | 機能有機化学 | 有機薄膜太陽電子材料の合成<br /><img src="/images/events/3/ni.png"  width="300px" />                     | 有機薄膜太陽電池 (OPV) は、柔らかい有機化合物を含むインクから印刷塗布することによって作製できるため、薄くて軽い次世代の太陽電池として注目されています。実際に、OPVに用いるインクから薄膜を作製し、発電する様子を観察してみましょう。                 |
